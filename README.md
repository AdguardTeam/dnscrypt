# DNSCrypt for Go

A modernization of [`github.com/ameshkov/dnscrypt`](https://pkg.go.dev/github.com/ameshkov/dnscrypt/v2), an implementation of the [DNSCrypt v2 protocol](https://dnscrypt.info/protocol).

## <a href="#api" id="api" name="api">Programming interface</a>

See [doc/api.md](./doc/api.md) for detailed programming interface documentation.

## <a href="#cli" id="cli" name="cli">Command-line interface</a>

DNSCrypt provides CLI tool that can work as a client or server. See [doc/cli.md](./doc/cli.md) for detailed documentation and examples.
